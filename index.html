Como o algoritmo funciona?

O Lumina AI usa HDR Fusion: cria uma versão super-exposta da sua foto via curva gamma e a funde com o original usando um mapa de luminância. Áreas escuras recebem mais luz; áreas claras ficam intactas. O Guided Filter garante bordas nítidas sem o halo característico.

100% local, 100% privado

Toda a lógica roda no seu navegador via Canvas API e JavaScript puro. O algoritmo é uma reimplementação fiel do código PyTorch original — mesma matemática, zero servidores externos. Sua foto nunca sai do dispositivo.
